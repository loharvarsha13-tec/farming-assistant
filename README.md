# 🌱 Voice Farming Assistant

## 📌 Overview
Voice Farming Assistant is an AI-powered web application designed to help farmers get instant agricultural advice using voice input. It supports local languages like Marathi and Hindi, making it accessible for non-technical users.

---

## 🎯 Problem Statement
Farmers often struggle to access timely and accurate agricultural advice due to:
- Lack of digital literacy
- Language barriers
- Limited access to experts

---

## 💡 Solution
This project provides a **voice-first interface** where farmers can:
- 🎤 Speak their problem
- 🤖 Get instant AI-based suggestions
- 🌍 Receive answers in local language
- 🔊 Listen to solutions via voice output

---

## 🚀 Features
- 🎤 Voice Input using Web Speech API  
- 🌾 Crop-specific Advice System  
- 🌍 Marathi & Hindi Language Support  
- 🔊 Text-to-Speech Output  
- 📱 Clean Card-based UI  

---

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- APIs: Web Speech API (Speech Recognition + Speech Synthesis)

---

## ⚙️ How It Works
1. User clicks 🎤 Speak button  
2. Speaks the farming issue  
3. Speech is converted to text  
4. Backend processes query  
5. Returns solution & tips  
6. Output is displayed + spoken
